Description \+:

This file defines the state machine of Cluedo game control. The node receives a random room position through /target topic. Then the robot reaches that room. Once the robot reaches the room, the node receives a hint. It repeats the process until a consistent hypothesis is found. The game ends when the correct hypothesis is found.
